## Project Overview
This project transforms raw eCommerce data from the `thelook_ecommerce` public dataset into a strategic decision-making tool
## Tech Stack
- **Database:** Google BigQuery (SQL)
- **BI Tool:** Google Looker Studio
- **Analytical Techniques:** RFM Segmentation, Margin Analysis, Time Series Forecasting.

##  Key Insights & Pages
1. **Product Analysis:** Identified high-margin categories (e.g., Suits & Coats with ~60% margin) vs. high-volume drivers.
2. **Customer Segmentation (RFM):** Grouped 60k+ users into "Champions", "At Risk", and "Lost" segments to optimize marketing ROI.
3. **Sales Trends:** Analyzed Month-over-Month (MoM) revenue growth and identified seasonal peaks.
4. **Global Sales Map:** Visualized geographical revenue distribution, identifying China and USA as primary markets.
